    Call chain: C++ ⇆ Java → JavaScript ⇆ Java ⇆ C++ → ⋯

    • The 2nd arrow comes from the Rhino used by the BlockLauncher, a JVM's JavaScript interpreter.

      The reverse arrow comes from the so-called interface dynamic proxy Proxy.newProxyInstance.
    • The 4th arrow comes from the System.load of the JVM and JNI.
